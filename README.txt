=========================================
Miki Portfolio — WordPress Theme
=========================================

HOW TO INSTALL
1. Log into your WordPress admin (yoursite.com/wp-admin)
2. Go to Appearance > Themes > Add New > Upload Theme
3. Choose the "miki-portfolio.zip" file and click "Install Now"
4. Click "Activate"

FIRST-TIME SETUP
- Three example projects are added automatically the first time you activate.
  Edit or add more from the "Projects" menu in the left sidebar.
- Go to Appearance > Customize to edit:
    - Hero name, role, tagline, and profile photo
    - About section text (education, experience, goals, interests)
    - Skills list (comma separated)
    - Contact links (email, Telegram, GitHub, LinkedIn)
- Add a profile photo: Appearance > Customize > Hero Section > Profile Photo
  (a placeholder image is included so the site isn't broken before you do this)

NOTES
- This theme is a single-page portfolio: Home / About / Skills / Projects / Contact
  are all sections on one page, linked via the top navigation.
- Projects are a custom post type, so you can add unlimited projects with a
  title, description, featured image, and a GitHub/live link — all from wp-admin,
  no code required.
- Works on any standard WordPress hosting (WordPress.org / self-hosted, or
  WordPress.com Business plan which allows theme uploads).
